# CT300 - Lab 1: Quản lý source code với Git

## Nội dung (theo PDF trang 1)
1. **Git là gì?** (trang 2): Hệ thống VCS phân tán để quản lý code.
1.1 **Lịch sử** (trang 2): Ra đời 2005, thay thế SVN.
1.2 **Mô hình** (trang 4): Local (working/staging/repo) + Remote.
2. **Sử dụng Git** (trang 8): Đã cài đặt, config, local/remote/branch.
3. **Công cụ đồ họa** (trang 15): GitHub web.

## Bài tập (trang 15–16)
- **Bài 1**: `web-project/index.html` + hotfix branch → merge.
- **Bài 2**: `c-project/hello.c` + sửa `\n` → hotfix.
- **Bài 3**: Push lên GitHub (PAT) – backup code.
- **Bài 4**: Clone vào `clone-test` → thêm `for.c` (in 1-10) → push.

## Kết quả
- Branches: `main` + 1 (hotfix).
- Commits: 2 (Bài 1+2, Bài 4).
- Chạy `for.c`: Copy code → [Replit](https://replit.com) → Output: 1\n2\n...\n10.
